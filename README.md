# vue-antd-admin
## 环境
* node -- 运行/编译
* yarn -- 依赖管理
* webpack -- 打包
* eslint -- 代码规约
* vue-cli -- 构建
## 依赖
* @antv/data-set: ^0.8.9
* ant-design-vue: ^1.0.3
* axios: ^0.18.0
* clipboard: ^2.0.1
* date-fns: ^1.29.0
* enquire.js: ^2.1.6
* mockjs: ^1.0.1-beta3
* pouchdb: ^7.0.0
* viser-vue: ^2.2.5
* vue: ^2.5.17
* vue-router: ^3.0.1
* vuedraggable: ^2.16.0
* vuex: ^3.0.1
## 安装
克隆项目到本地:
```
$ git clone https://github.com/iczer/vue-antd-admin.git
```
安装依赖：
```
$ npm install
```
## 运输技能开发环境
```
$ npm run dev  --host=devRioh
```
## 打包运输技能
```
$ npm run build  --host=betaRioh
```
